## If first time to PUSH and COMMIT

```bash
  git clone https://github.com/kenaniscoding/cpecog2_paper.git
```
## How to PUSH and COMMIT changes 
Add File
```bash
  git add .
```
Commit with a MESSAGE
```bash
  git commit -m "message"
```
Push to the branch
```bash
  git push
```
